================================================================================
KEY PRINCIPLES
================================================================================

✅ DIFFERENT THRESHOLDS FOR DIFFERENT CONTRIBUTION TYPES
   - Strategic (PLIPs): 10 / 3-9 / 1-2
   - Implementation (PRs): 300 / 100-299 / 1-99
   - No arithmetic equivalence needed

✅ SAME TIER = SAME PRESTIGE
   - Platinum Strategic = Platinum Implementation (both are core leaders)
   - Different paths, equal recognition

✅ ROLE-BASED, NOT ARITHMETIC
   - "What defines a core leader?" not "What equals X PLIPs?"
   - Thresholds create similar-sized leadership groups

✅ SIMILAR ABSOLUTE NUMBERS AT EACH TIER
   - Platinum: 8 strategic + 12 implementation = ~20 core leaders
   - Gold: 22 strategic + 17 implementation = ~40 active contributors
   - Goal achieved: manageable tier sizes

✅ PERCENTAGE DIFFERENCES ARE OK
   - Different pool sizes (63 vs 382)
   - Different barriers to entry
   - Strategic work more specialized, PR work more accessible

================================================================================
WHY THIS WORKS
================================================================================

Problem: "How many PRs equal one PLIP?"
Answer: Wrong question. Ask instead: "What defines leadership in each domain?"

Strategic Leadership (PLIPs):
  • 10+ PLIPs over years = proven architectural vision
  • Requires: strategic thinking, community consensus, long-term planning
  • Pool: 63 people (specialized skill)

Implementation Leadership (PRs):
  • 300+ PRs over 3 years = proven execution excellence
  • Requires: code quality, maintenance, continuous delivery
  • Pool: 382 people (broader accessibility)

Result: Different thresholds, equivalent community impact
